4. **Fāze 3 (generate.py šķelšana) — PILNĪGI PAVEIKTA 2026-04-30 (PR #5–#16 + F3g closure).**
   - **Trajectory:** `src/generate.py` 4250 → 173 LOC (-96% — kā re-export shim). 17 src/render/* moduļi (1 leaf `_common` + 1 orchestrator + 15 sub-pages).
   - **F3a-F3e + F3f.2 + F3f.3 + F3f.5 + F3f.4 + F3f.1:** sub-page carve-outs ar byte-identitāti.
   - **F3g closure (PR #17, 2026-04-30):**
     - `_orchestrator.py` jauns — owns `generate_public_site` + `_generate_sitemap` + `_generate_og_image`.
     - `src/render/__init__.py` re-eksportē — kanoniskais `from src.render import generate_public_site`.

     - `_load_wiki_profile` promoted no analyses.py uz _common.py (politicians-concern, ne analyses-concern).

5. **F3 implementation lessons (mācība F4+ un nākamajām paketēm):**
   - **Pre-promote shared leaf helpers BEFORE writing peer sub-pages.** F3-prep + F3b + F3d + F3f.5 disciplīna kept the cycle closed.
   - **`src/render/__init__.py` apzināti tukšs līdz F3g** — pirms tam re-eksports radītu cycle ar pre-shim `src/generate.py`. Post-F3g bug-free, jo `src/generate.py` ir tīrs shim.
   - **Re-export shim from generate.py is wide, not narrow.** Tests + agents importē ~85 privātus simbolus; F3 phases progresīvi paplašināja shim.
   - **Char fixture pattern** (`tests/test_render_chars.py` + 10 `tests/fixtures/render_baseline_*.json`) — session-scoped `generate_public_site(output_dir=tmp)` + SHA-256 hashes. REGEN=1 bootstraps; bare run asserts. Determinism guaranteed by `ATMINA_ASSETS_VERSION="test"` pin.
   - **Self-contained vs pass-through orchestrators.** Self-contained when data not reused downstream (`render_personas`, `render_news`, `render_positions`, `render_bills`, `render_laws`, `render_x`). Pass-through when caller pre-fetched data shared across multiple consumers (`render_contradictions`, `render_politicians`, `render_parties` post-F3g.2, `render_votes`, `render_tensions` + `render_links`, `render_dashboard`, `render_blog`, `render_analyses`, `render_syntheses`).

6. **`src.analyze` paliek neaiztikts** — 3 aģentu prompti (`claim-extractor`, `video-extractor`) importē 4 publiskās funkcijas. Tas nav refaktoringa plānā šobrīd, bet jāatzīmē kā high-stable API ja vēlāk nāks `src/analysis/` pakete.
